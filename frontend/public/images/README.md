- `cactus1.png` - Cactus variant 1
- `cactus2.png` - Cactus variant 2  
- `cactus3.png` - Cactus variant 3
- `bone-bronze.png` - Bronze bone collectible
- `bone-silver.png` - Silver bone collectible
- `bone-gold.png` - Gold bone collectible
- `coconut.png` - Glowing coconut collectible

The game will use these EXACT images without any distortion or structural changes.
