SPLASH SCREEN IMAGE INSTRUCTIONS
================================

To add your startup image for the splash screen:

1. Place your image file in this folder (frontend/public/)
2. Name it: splash-image.png
3. Supported formats: PNG, JPG, WEBP

The splash screen will automatically:
- Display your image when users first enter the app
- Show for 3 seconds, or users can click to skip
- Fallback to text logo if image not found
- Work with both light and dark themes

RECOMMENDED IMAGE SPECS:
- Format: PNG or JPG
- Size: 1200x800px or similar aspect ratio
- File size: Under 500KB for fast loading
- Background: Transparent PNG works best, or match app theme colors

Example path:
frontend/public/splash-image.png
